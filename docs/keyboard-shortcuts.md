# Keyboard shortcuts

| Shortcut | Action |
| --- | --- |
| `Ctrl/Cmd+K` | Focus the assistant input |
| `Alt+2` | Open the Tools tab |
| `Alt+3` | Open the Library tab |
| `Ctrl/Cmd+Shift+E` | Export the current transcript |
| `Ctrl/Cmd+Shift+Backspace` | Clear the conversation |
| `Enter` | Send a message |
| `Shift+Enter` | Insert a new line |

Browser or operating-system shortcuts may take precedence on some devices.
