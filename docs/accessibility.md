# Accessibility checklist

## Keyboard

- Tab order follows the visible interface.
- All tabs, workflow chips, forms, source links and dialog controls are reachable.
- `Ctrl/Cmd+K` focuses the assistant input.
- `Alt+2` opens Tools; `Alt+3` opens Library.
- `Ctrl/Cmd+Shift+E` exports the transcript.
- `Ctrl/Cmd+Shift+Backspace` clears the conversation.

## Screen readers

- Tabs expose their selected state and controlled panels.
- Conversation additions and toast messages use polite live regions.
- Inputs have programmatic labels.
- Icon-only controls have accessible names.

## Visual

- Light and dark themes maintain visible focus and contrast.
- Information does not rely only on colour.
- Reduced-motion preferences disable animations and smooth scrolling.
