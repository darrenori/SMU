# Changelog

All notable project changes are recorded here.

## 2.0.0

- Added nine deterministic legal-workflow frameworks.
- Added local matter metadata and persistent conversation history.
- Added transcript export and per-response copy controls.
- Added a planning-date calculator with calendar and weekday modes.
- Added chronology sorting and CSV export, deterministic line comparison and document text inspection.
- Added local privacy detection and redaction for common identifiers.
- Added a five-point review checklist.
- Added searchable official Singapore legal sources and a legal glossary.
- Added light, dark and system themes plus keyboard shortcuts.
- Added automated tests, manifest verification and extension packaging.
- Confirmed a zero-permission, no-network extension architecture.

## 1.0.0

- Rebuilt the original hackathon interface as a deployable product site and Manifest V3 extension prototype.
